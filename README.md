# Прогрессивный метроном для гитары

Тренировочный инструмент в одном HTML-файле: запускается локально в браузере, не требует установки, интернета и зависимостей. Метроном начинает с медленного темпа и автоматически разгоняется через заданные интервалы — удобно для отработки техники, гамм и упражнений на скорость.

## Возможности

- Точное время благодаря Web Audio API (lookahead-планировщик), без дрейфа `setInterval`.
- Три параметра разгона: стартовый темп, шаг увеличения и **размер партии в ударах** (с учётом такта).
- **Размер такта с акцентом на 1-ю долю**: без акцента, 2/4, 3/4, 4/4, 6/8, 8/8.
- **Четыре тембра клика**: Click (синус), Woodblock (деревянный), Beep (электронный), Click Tap (мягкий шум).
- **Регулировка громкости** через мастер-канал (0–100%).
- **Голосовой отсчёт** перед стартом: «раз, два, три, четыре» в темпе на русском (через SpeechSynthesis API браузера).
- **Пресеты в localStorage**: три встроенных + неограниченное число пользовательских с возможностью загрузки и удаления.
- **Автосохранение настроек**: при следующем открытии всё восстанавливается.
- Большой счётчик текущего BPM с пульсирующей анимацией на каждый удар (акцент подсвечивается тёплым оттенком).
- Точечный индикатор долей такта в реальном времени.
- Прогресс-бар и таймер MM:SS до следующего ускорения, плюс подсказка «Следующий BPM».
- Старт / Пауза / Сброс. Управление пробелом с клавиатуры.
- Современный тёмный дизайн в стиле glassmorphism, адаптивный под мобильные и десктоп экраны.
- Поддержка `prefers-reduced-motion` для пользователей с чувствительностью к анимациям.

## Запуск

1. Откройте файл `metronome.html` двойным кликом — он откроется в браузере по умолчанию.
2. При первом нажатии на «Старт» браузер инициализирует звук (так требует Web Audio API).
3. Громкость регулируется встроенным ползунком и системным микшером.

Поддерживаемые браузеры: Chrome, Edge, Firefox, Safari (актуальные версии). Работает на iOS Safari и Chrome на Android. Для голосового отсчёта на русском требуется системный TTS-голос (на большинстве ОС встроен).

## Параметры темпа и разгона

| Поле | Назначение | Диапазон | Шаг по умолчанию |
|------|------------|----------|------------------|
| **A** | Стартовый темп, ударов в минуту | 20 – 300 BPM | 1 |
| **B** | Шаг увеличения BPM в начале каждой новой партии | 1 – 30 BPM | 1 |
| **C** | Размер партии — количество ударов до ускорения | 4 – 1024 уд. | 4 |

Каждое поле имеет ползунок и числовое поле — можно править как мышью, так и вводить значение с клавиатуры. Под ползунком **C** видна подсказка с эквивалентом в тактах: например, при размере 4/4 и C = 120 это «≈ 30 тактов».

### Учёт тактов при ускорении

Ускорение всегда происходит на **границе такта** — на 1-й доле. Если значение **C** не делится нацело на размер такта, оно округляется вверх до ближайшего целого такта. Примеры при размере 4/4:

- C = 120 → 30 тактов = 120 ударов (точно).
- C = 130 → округлится до 132 ударов = 33 такта.
- C = 50 → округлится до 52 ударов = 13 тактов.

В режиме «Без акцента» ускорение происходит ровно через **C** ударов.

## Параметры звука и такта

| Параметр | Опции |
|----------|-------|
| Размер такта | Без акцента, 2/4, 3/4, 4/4, 6/8, 8/8 |
| Тембр клика | Click, Woodblock, Beep, Click Tap |
| Громкость | 0–100% |
| Голосовой отсчёт | Вкл / Выкл |

Тембр прослушивается одним кликом — при выборе на остановленном метрономе сразу проигрывается короткое превью.

## Пресеты

Встроенные пресеты:

- **Новичок 60→120** — 60 BPM, шаг 5, партия 120 уд. (30 тактов 4/4), Click
- **Скорость 80→160** — 80 BPM, шаг 8, партия 64 уд. (16 тактов 4/4), Woodblock
- **Разогрев 40→100** — 40 BPM, шаг 3, партия 96 уд. (32 такта 3/4), Tap

Пользовательский пресет сохраняет: A, B, C, размер такта, тембр, громкость. Хранилище — `localStorage` браузера (ключ `metronome_presets_v2`). Чтобы перенести пресеты на другое устройство, можно скопировать значение этого ключа через DevTools.

## Логика работы

1. Нажимаете «Старт» — если включён голосовой отсчёт, проигрывается «1, 2, 3, 4» в текущем темпе, а затем метроном стартует с темпом из поля **A**.
2. Запускается счётчик ударов в текущей партии.
3. После того как сыграно **C** ударов и завершён очередной такт — текущий темп увеличивается на **B** BPM, счётчик сбрасывается и стартует следующая партия.
4. Цикл повторяется до достижения потолка 300 BPM или до нажатия «Стоп»/«Сброс».

Пример: A = 60, B = 5, C = 120, размер 4/4. Первые 30 тактов (120 ударов) — 60 BPM, следующие 30 тактов — 65 BPM, далее 70 BPM и так далее. Каждый 1-й удар такта акцентируется тембром повыше и тёплой подсветкой пульса. На дисплее «До ускорения» видно, сколько ударов осталось до следующего повышения.

## Управление

| Действие | Кнопка / Клавиша |
|----------|------------------|
| Старт / Пауза | Кнопка «Старт» или `Пробел` |
| Стоп без сброса прогресса | Кнопка «Стоп» |
| Полный сброс к стартовому BPM | Кнопка «Сброс» |
| Загрузить пресет | Кнопка «Загрузить» рядом с пресетом |
| Сохранить пресет | Ввести имя и нажать «Сохранить» (или Enter) |
| Удалить пользовательский пресет | Кнопка `✕` |

Изменение **B** и **C** во время работы применится со следующей партии. Изменение **A** во время работы не влияет на текущий темп — оно вступит в силу после нажатия «Сброс». Загрузка пресета во время работы автоматически сбрасывает метроном.

## Звук

Все звуки синтезируются на лету через Web Audio API — никаких внешних файлов:

- **Click** — чистый синусоидальный «тик» с быстрой огибающей.
- **Woodblock** — триангл через bandpass-фильтр с резким pitch-down.
- **Beep** — square-волна, более «электронный» характер.
- **Click Tap** — фильтрованный белый шум, мягкий перкуссионный звук.

Акцент на 1-й доле: на октаву выше и громче основного клика. Громкость регулируется единым мастер-гейном.

## Голосовой отсчёт

При включённом тоггле перед стартом метронома произносятся слова «раз, два, три, четыре» в текущем темпе через `window.speechSynthesis`. Если в системе нет русского голоса — произнесение может прозвучать на английском или быть пропущено, но синхронизирующий клик всё равно звучит и большая цифра отсчёта показывается на экране. Отсчёт срабатывает только при первом старте; продолжение после паузы — мгновенное.

## Хранение данных

Всё хранится локально в `localStorage` вашего браузера, ничего не отправляется в сеть:

- `metronome_settings_v2` — последние выбранные настройки (восстанавливаются при перезагрузке).
- `metronome_presets_v2` — пользовательские пресеты.

Чтобы очистить — DevTools → Application → Local Storage → удалить ключи.
